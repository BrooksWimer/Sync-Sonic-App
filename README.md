This is an [Expo](https://expo.dev) project that functions as the controller for the physical Sync-Sonic device. At this time the device is a Raspberry Pi with custom hardware and software to allow for playing music across multiple bluetooth devices at once. Find more information on the physical device [here](https://github.com/BrooksWimer/SyncSonicPi). The goal is to get rid of brand isolation and be able to play the same music across speakers from any brand across the house, yard, or party!

## Getting Started
1. Clone the repository
   ```bash
   git clone https://github.com/BrooksWimer/Sync-Sonic-App
   cd Sync-Sonic-App
   ```
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
    npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo
